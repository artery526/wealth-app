操作記錄
Apps Script 部署說明

1. 打開您的 Google Sheets → 點選「擴充功能」→「Apps Script」

2. 貼上軍師提供的 Apps Script 程式碼

3. 點選「部署」→「新增部署作業」→ 類型選「網路應用程式」

4. 執行身分選「我」，存取權選「所有人」

5. 點「部署」後複製 Web App URL 貼入上方欄位
